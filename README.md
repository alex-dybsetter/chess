# Chess
A simple 2 player chess app.

Technologies used: 
* Java
* Kotlin
* MVP Design Pattern
* Firebase
    - Cloud Firestore
* RxJava
* ButterKnife
* MVP Design
* Events
* Fragments
* Custom Adapters
